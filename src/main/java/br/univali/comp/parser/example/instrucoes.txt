Instalar Java jdk14

Instalar javaCC : Download, setting on windows:
https://javacc.github.io/javacc/#getting-started

Para executar a anpálise léxica:
no package parser existe o arquivo Tokeniser.jj Após cada ajuste nesse arquivo é necessário mandar gerar seus arquivos ->
# executar: javacc Tokeniser.jj
# executar: javac ImpCA_Lex.java
Após isso para testar um arquivo com uma sequencia de caracteres segue o exemplo.
# executar: java ImpCA_Lex example.imp
Para executar o código no intelij será necessário setar o package para os arquvios que forem gerados.
